Welcome to python-livy's documentation!
=======================================

*python-livy* is a lightweight tool to interact with `Apache Livy <https://livy.incubator.apache.org/>`_. Provide both CLI tools and library that works nicely in pure and native Python.

.. image:: ../screenshot.png

This tool is originally designed for task submission and script debugging. If you do have an idea on how to make this tool for powerful, please create a issue to share it with me :)
