Welcome to python-livy’s documentation!¶

python-livy is a lightweight tool to interact with Apache Livy. Provide both CLI tools and library that works nicely in pure and native Python.

This tool is originally designed for task submission and script debugging. If you do have an idea on how to make this tool more powerful, please create a issue and share it with me :)
